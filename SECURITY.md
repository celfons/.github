# Security Governance

This file is the entry point to security governance for the `celfons` organization.

All security rules, remediation guidelines, and tool-specific interpretation live inside this repository.

- Vulnerability remediation rules: [`SECURITY/RULES/`](SECURITY/RULES/)
- Tool-specific guidance: [`SECURITY/TOOLS/`](SECURITY/TOOLS/)
- Security philosophy and overview: [`SECURITY/OVERVIEW.md`](SECURITY/OVERVIEW.md)

Do not duplicate rules from this repository in product repositories.
Do not report vulnerabilities by opening issues here directly — use the automated pipeline.
